# Проект классификации изображений CIFAR-10

Этот проект реализует свёрточную нейронную сеть (CNN) для классификации изображений из набора данных CIFAR-10 с использованием PyTorch.

## Структура проекта

```
py_src/project/cifar10/
├── config.py             # Параметры конфигурации
├── data_work.py          # Загрузка и предварительная обработка данных
├── main.py               # Основной исполняемый скрипт
├── requirements.txt      # Зависимости Python
├── test_model.py         # Функции тестирования модели
├── train_model.py        # Функции обучения модели
├── data/                 # Хранение набора данных
│   ├── cifar-10-python.tar.gz
│   └── cifar-10-batches-py/
└── net/
    └── CNN.py            # Определение архитектуры CNN
```

## Архитектура модели

CNN состоит из:
- **Слоёв извлечения признаков** (`self.features`):
  - 5 свёрточных слоёв с пакетной нормализацией, активацией ReLU и максимальным пулингом
  - Обрабатывает 3-канальные (RGB) изображения 32×32
  - Использует ядра 3×3 с padding=1 для сохранения пространственных размеров
  - Максимальный пулинг (2×2) уменьшает пространственные размеры вдвое

- **Классификационные слои** (`self.classifier`):
  - Два полносвязных слоя
  - Dropout (0.5) для регуляризации
  - Выходной слой с 10 классами

## Процесс обучения

- **Гиперпараметры**:
  - Эпохи: 10
  - Скорость обучения: 0.001
  - Оптимизатор: Adam
  - Функция потерь: CrossEntropyLoss
  - Устройство: GPU (CUDA), если доступно, иначе CPU

- **Рабочий процесс обучения**:
  1. Модель обучается указанное количество эпох
  2. Для каждого батча:
     - Прямой проход
     - Вычисление потерь
     - Обратный проход
     - Обновление весов
  3. Точность и потери отображаются для каждой эпохи

## Тестирование и оценка

Модель оценивается по следующим критериям:
- **Общая точность**: Процент правильно классифицированных изображений
- **Средняя уверенность**: Средняя вероятность softmax для правильных предсказаний
- **Взвешенная оценка**: Общая точность × средняя уверенность

## Использование

1. Убедитесь, что все зависимости установлены:
   ```bash
   pip install -r requirements.txt
   ```

2. Запустите основной скрипт:
   ```bash
   python main.py
   ```

3. Чтобы использовать предобученную модель, установите `LEARN = False` в `main.py` и убедитесь, что файл `cnn_model.pth` доступен.
